📖 วิธีสร้าง Google Apps Script

ขั้นตอนการสร้าง Google Apps Script Web App:

  1. เปิด script.google.com → สร้างโปรเจกต์ใหม่
  2. ลบโค้ดเดิมออก แล้วแทนที่ด้วยโค้ดด้านล่าง
  3. แก้ไข SHEET_ID ให้ตรงกับ Google Sheet ID
  4. บันทึก → Deploy → New Deployment → Web App
  5. ตั้งค่า Execute as: Me, Who has access: Anyone
  6. คัดลอก URL มาวางในช่องด้านบน
